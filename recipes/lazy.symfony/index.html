Lazy - Symfony

  • Requirements
  • Setup

Migrations

If you're coming from a previous version of the recipe or another recipe, you might be interested reading the following migration guides:
  • Makefile variables refactoring - 2023-02

Requirements

  • Manala CLI to update the recipe
  • Make
  • Docker Desktop 2.2.0+ or Docker Engine + Docker Compose

Setup

Ok, let's init a foo symfony demo application:

manala init -i lazy.symfony foo
cd foo

Up the system and open a shell into an another terminal:

make up
make sh

Create the foo symfony demo application into /tmp and move files into your project (symfony can't create application in a non-empty dir):

symfony new /tmp/foo --no-git --demo
rsync -a /tmp/foo/ ./

Browse http://localhost:8080 and enjoy :)
